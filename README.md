# http-airsensor
Collects the following data and exposes it through an HTTP server:
* Temperature
* Humidity
* Particulate Matter

TBD:
* Pressure
* Carbon Monoxide


## Sensors supported
* HDC1080
* PMS3003
* Rasberry Pi CPU Temperature

TBD:
* BME280
* ZE07-CO

Potential:
* DHT22
* BMP180
* MQ-7
* MICS-5524

These sensors can easily be found on Aliexpress (don't forget to choose a seller with e-packet shipping)
